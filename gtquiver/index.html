GTQuiver 箭袋与自定义箭 ​

作者：RusBrook
插件定制：1415375222
GT 系列插件交流群：985955382

GTQuiver 兼容 1.12.2 - 1.21.11。箭袋内容跟随物品保存，可用于生存服、RPG 服和跨服。

依赖 ​

项目说明
必装GTCore
可选PlaceholderAPI、CraneAttribute、AttributePlus、MythicMobs、GTCore 已接入的物品库
Java插件兼容 Java 8，Java 版本仍需满足当前服务端要求

插件无需授权配置。安装顺序为 GTCore、GTQuiver，依赖和 Jar 有变动时完整重启服务器。

配置文件 ​

text
plugins/GTQuiver/
├─ config.yml
├─ quivers.yml
├─ arrows.yml
├─ messages.yml
└─ player-settings.yml
文件内容
config.yml装备槽、无限附魔、箭回收、属性插件、Lore、GUI、粒子
quivers.yml箭袋外观、容量、权限、条件、存入规则
arrows.yml自定义箭物品和命中效果
messages.yml提示消息
player-settings.yml玩家执行自动收箭命令后生成

修改配置后执行：

text
/gtq reload

配置有误时不会覆盖当前正在使用的配置，具体原因查看控制台。

发放 ​

text
/gtq give <玩家> <箭袋ID> [数量] [bound|unbound]
/gtq givearrow <玩家> <箭ID> [数量]

默认配置带有：

text
箭袋：survival、ranger
自定义箭：flame、frost

查看当前已加载 ID：

text
/gtq list

绑定箭袋：

text
/gtq give 玩家 survival 1 bound

非绑定箭袋：

text
/gtq give 玩家 survival 1 unbound

不写模式时默认为非绑定。绑定箭袋只能由绑定玩家拾取和使用，其他玩家不能打开、切箭、射击或作为自动收箭目标。

玩家操作 ​

打开箭袋 ​

主手拿箭袋：

text
Shift + 右键空气或方块

副手拿箭袋不会打开 GUI。

装备箭袋 ​

默认装备位置是副手：

yaml
settings:
  equipped-slots:
    - "OFF_HAND"

也可以改成快捷栏或背包槽位，详见 config.yml。

切换箭矢 ​

装备箭袋后：

text
Shift + 左键空气或方块

当前箭种会移到其他箭种后面，下一种箭成为首选箭。同一种完整物品会合并为一个箭种，不按堆叠数量重复切换。

射箭规则 ​

  • 背包已有兼容箭时，按原版规则处理。
  • 背包没有兼容箭时，从装备箭袋中取箭。
  • 箭袋按 GUI 从左到右消耗。
  • 当前箭用完后自动切换下一种。
  • 普通弓支持普通箭、药水箭、光灵箭。
  • 弩额外支持烟花火箭。
  • 无限附魔默认只对无名称、无 Lore、无属性数据的普通原版箭生效。
  • 创造模式默认不消耗箭袋中的箭。

箭袋 Lore 会显示总数量、占用槽位、当前箭和箭种明细。

箭回收 ​

默认禁止拾取射出的箭：

yaml
arrow-recovery:
  allow-non-infinity: false

改为 true 后，实际消耗且不是无限弓射出的箭可以拾取。玩家装备了可接收该箭的箭袋时，拾取会直接存入箭袋；箭袋放不下时进入普通背包。

自动收箭默认开启。有权限的玩家可自行切换：

text
/gtq autopickup on
/gtq autopickup off

权限：

text
gtquiver.command.autopickup

该权限默认不授予。

PlaceholderAPI ​

text
%gtquiver_current_arrow%
%gtquiver_current_arrow_colored%

第一个返回去色名称，第二个保留物品颜色。没有装备箭袋或箭袋为空时返回 未选择。
